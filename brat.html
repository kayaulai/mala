brat

Web app with a flexible schema focused on structured prediction tasks in NLP, including NER, coreference, UD syntax, and others.

Release year: 2012

Paper: https://aclanthology.org/E12-2021/

Code: https://github.com/nlplab/brat

Availability: open-source

Data types: tags, chunk-spans, overlapping-spans, dependency-tree, graph

Import formats: plaintext, proprietary

Export formats: plaintext, proprietary

Keywords: ner, coreference, ud
