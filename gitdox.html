GitDox

Multilayer annotation editor featuring GitHub integration, realtime collaborative editing, and validation features.

Release year: 2017

Paper: http://corpling.uis.georgetown.edu/amir/pdf/GitDOX_A_Linked_Version_Controlled_Online_XML_Editor_prepub.pdf

Code: https://corpling.uis.georgetown.edu/gitdox/

Availability: open-source

Data types: tei, tags, span-labels, nested-span-labels

Import formats: xml, plaintext

Export formats: xml, csv

Keywords: multilayer, semantics, coreference
